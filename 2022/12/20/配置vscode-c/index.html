配置Vscode C++

2022/12/20

工作环境：

windows10，msvc

1. 下载并安装

网址：Visual Studio Code - Code Editing. Redefined

下载后打开安装，建议以管理员身份运行，弹窗不用管。

安装路径需要注意一下，默认是安装到AppData里，可以改到program files或者自己想安装的位置。

建议将上图中的其他中的四个全部勾选

接下来等待安装即可。

2. 配置中文

Ctrl+Shift+P

输入Configure Display Language, Enter

选择中文，Enter，重启软件即可

*.完全卸载

删除C:\Users\用户名\.vscode文件夹

删除C:\Users\用户名\AppData\Roaming\Code文件夹（这个我不知道需不需要，并未试验过）

3. 配置C++

选择一个配置c++的文件夹，vscode会在配置后在此文件夹中生成一些文件。若是以后仍需要c++环境，复制该文件夹即可。

在该文件夹右击选择在code中打开。

3.1 安装c++编译器

下载安装MSYS2

原网站下载可能需要开个vpn

网盘链接：https://pan.baidu.com/s/1oKfC7J2733VNMZvgJwYt9g?pwd=g0xs
提取码：g0xs

  • 问题

    若是安装卡在中间不动了，点击详细信息，显示Updating trust database…

  • 解决

    点击取消并重新安装即可，注意不要管该文件夹不是空的的警告

打开刚刚安装的msys2，输入pacman -Sy更新msys2，再输入pacman -Su gcc安装gcc

对于Proceed with installation? [Y/n]输入y即可

  • 问题

    fork:Resource temporarily unavailable

    原因是对进程的数量进行了限制，可以通过

    ulimit -u来查看最大进程数，若是小于1000，估摸应该会很容易出现这个错误

  • 解决

    通过ulimit -u 数字(建议65535)来修改最大进程数

    但我由于好像不是root权限失败了

    最终通过重启msys2来清楚所有进程来空出进程来安装，再次运行上述安装命令即可

3.2 验证是否安装成功

gcc -v

输出版本号

3.3 在vscode中继续配置

在vscode中打开前面创建的文件夹

在扩展左侧栏第6个按钮(四个方块的图标)中安装c/c++

新建hello.cpp文件

使用Ctrl+`来新建一个终端Terminal

  • 问题

    无法打开，error code:740

  • 解决

    在设置里的终端里Intergrated > default profile: windows设置终端使用powershell或cmd

    最后在Reload Window一下

3.4 测试是否配置成功

g++ hello.cpp

此时该文件夹下会出现一个a.exe文件

./a.exe

输出两行都是hello

似乎好像就成功运行了emmmm

但是又没有太成功，因为我安装过vs2022使得它已经装过编译器了emmm

4. 官方方式

调用命令面板，输入c/c++选择Edit configurations(UI)更改编译器路径，选择安装的编译器的路径

发现在文件夹下多出一个.vscode文件夹，里面有一个配置文件，即成功

重复上述验证步骤即可

5. 快速生成

调用命令面板，输入task，选择

然后选择你安装的编译器

会发现.vscode文件夹下生成了task.json

此时再在.cpp文件夹下使用Ctrl+Shift+B即可快捷对.cpp文件进行编译

物理层

基本概念

  1. 作用：屏蔽掉传输媒体和通信手段的差异
  2. 特性：
    1. 机械特性
    2. 电气特性
    3. 功能特性
    4. 过程特性
  3. 穿行传输

基础知识

  1. 数据通信系统的模型

    1. 源系统
      1. 源点：产生要传输的数据
      2. 发送器：将数字比特流进行编码
    2. 传输系统
    3. 目的系统
      1. 接收器：对数据进行逆编码
      2. 终点：接收并输出

    通信的目的是传送消息，而数据是消息的实体，而数据就是信号的序列。

    1. 模拟信号（连续信号）：取值是连续的
    2. 数字信号（离散信号）：取值是离散的

    代表不同离散数字的基本波形就称为码元，如二进制编码就有两种不同的码元。

    波特就是码元每秒。如果一个码元能够有k种情况，那么他就能携带log2K个bit。

  2. 信道的基本概念

    信道：向某一个方向传送信息的媒体

    几种不同的通信

    1. 单向（工）通信：只有一个方向的通信
    2. 双向交替通信：半双工通信，双方都可发，但不可同时发
    3. 双向同时通信：全双工通信，双方都可发，且可同时发

    基带信号：来自源的信号

    包含较多的低频，不好进行传输，因此要进行调制。

    1. 基带调制（编码）：仅对波形进行变换

      常用编码方式：

      1. 不归零制（没有自同步）：正电平1，负电平0
      2. 归零制：正脉冲1，负脉冲0
      3. 曼彻斯特编码（拥有自同步）：周期中间向上跳变代表0，向下跳变代表1
      4. 差分曼彻斯特编码：边界开始有跳变代表0，无跳变代表1

    2. 带通调制：使用载波进行调制，将其频率范围搬移到较高的频段，转换为模拟信号。（带通信号）

      基本带通调制方法：

      1. 调幅（AM）：振幅随数字信号变化
      2. 调频（FM）：频率随数字信号变化
      3. 调相（PM）：初始相位随数字信号比

  3. 信道的极限容量

    只要数字信号的失真不是非常严重，就可以识别回来。

    因此，极限容量就是看数字信号的失真情况

    1. 信道能通过的频率范围

      码间干扰：当频率较高时，失真会导致其陡峭不明显。

      无噪声情况下的极限波特率：奈氏准则：当不考虑噪声时，由于码间干扰，最高速率为2W波特，如果带宽为W（应该继续研究编码）

    2. 信噪比

      信号中会产生随机噪声，影响识别。

      信噪比：信号的平均功率噪声的平均功率\frac{信号的平均功率}{噪声的平均功率}噪声的平均功率信号的平均功率​(记做S/N)

      信噪比(dB)=10log10(S/N)=10log_{10}(S/N)=10log10​(S/N)

      有噪声情况下的极限比特率：香农公式（达不到，因为还有其他因素）：极限信息传输速率C=Wlog2(1+S/N)C=Wlog_2(1+S/N)C=Wlog2​(1+S/N)比特（不管怎么编码，都无法突破）

    但通过让码元能够携带更过比特的信息来提高传输速率。但是这也是有上限的，因为有噪声干扰，可能无法分辨码元。

物理层下面的传输媒体

  1. 导引型传输媒体：电磁波被导引沿固体媒体传播

    1. 双绞线（无屏蔽双绞线）

      将两根弦放在一起会有电磁干扰，而绞合能减少这一点。绞合度越高，电磁干扰越小。

      屏蔽双绞线：通过给每一根双绞线或者整条电缆增加屏蔽层，可以提高数据传输速率。

    2. 同轴电缆

      主要用于有线电视网的小区中。抗干扰很高，能传输较高速率数据。

    3. 光缆（损耗小，经济，抗干扰好，体积小）

      通过让入射角足够大，产生全反射来进行引导。

      多模光纤：以不同的入射角射多条光线。

      单模光纤：当为一根，且直径足够小的时候，就不会产生反射，损耗也较小。

      可以使用中继器来延长传输，但是不能无限串联，集线器也是一样，来无线扩大范围。

    4. 架空明线

      直接在电线杆上架设相互绝缘的电线

  2. 非导引型传输媒体

    1. 无线传输可以使用各个波段进行通信。

      ELF，SLF，ULF，VLF，LF，MF，HF，VF，UF，SF，EF，TF。

      E(Extremely)，S(Super)，U(Ultra)，V(Very)

    2. 无线电微波通信

      较短且为直线，因此距离较短。频率高，信道容量大，质量高。

      多径效应：在经过几次反射后有多条路径合并会造成很大的失真。

      微波接力：为了远距离传输，就必须建立中继站，将接收到的信号放大

      缺点：相邻站需要直视，会受到恶劣天气影响，维护成本高。

    3. 短波通信：靠电离层反射，也会产生多径效应

    4. 卫星通信：通信距离远，但具有较大的传播时延

    不管哪种非引导，都必须保证误码率不大于可容许的范围。

    信噪比越大，调制技术的数据率越小，误码率越小。而对于我们平常使用的移动设备，会根据当前情况自动调整调制技术。

信道复用技术

复用（可能是一个也可能是多个用户）：多条信道合并为一条信道进行公用。通过一个复用器和分用器即可达到效果。

  1. 频分复用FDM：在同一条信道中，可以以不同的频率同时传输多条信息（即占用不同的带宽资源）。宽带变大。

  2. 时分复用TDM：在同一条信道中，按一定长的TDM帧的时间进行划分，在每个周期内，进行划分序号，每个对应的序号的时间内只传输对应用户的信息。宽带不变。

    TDM信号：等时信号

  3. FDMA，TDMA：频分/时分多址接入，能让多个用户共用一个信道。

  4. STDM：统计时分复用。由于计算机的突发性，而TDM又是非常固定的，因此时分复用会导致信道利用率极低。STDM会将受到的不同信道的信息缓存起来，当要输出一个信息的时候，循环对信道进行扫描，跳过空的信道，将有缓存的进行输出。

    由于不固定，因此传输的时候还要加上用户信息。

  5. 波分复用WDM：光的频分复用。DWDM，密集波分复用（指复用了很多波长的WDM）。

    其对应的复用器和分用器称为光复用器（合波器）和光分用器（分波器）。

    光的传输是会逐渐衰减的，因此每隔一段时间就必须用EDFA(一种放大器)，对其进行放大。

  6. 码分复用CDM：给每一个用户分配不同的码型，数据为1的时候就是这个码型，0就是对应的反码，最终将各个用户的结果相加即得到最终的结果。只要保证每个用户的码型都是互相正交的，也就是内积为0，最终在与自己的码型内积后即可获取自身的这一条序列。

    一个码型由多个01组成，每一个占用的时间被称为一个码片。保证所有的码片序列是同步的。

    通过这种方式，实际上是提高了频带宽度，也就做到了扩频。这种方式是直接序列扩频DSSS。

数字传输系统

  1. 早期问题：

    1. 速率标准不统一
    2. 不是同步传输，给复用带来了很大的麻烦

  2. 1988年推出了同步光纤网SONET，以51.840Mbit/s为基础，对电信号称为第一级同步传送信号，对光信号称为第一级光载波。

    而后SDH基于SONET，速率为155.520Mbit/s，称为第一级同步传递模块。

宽带接入技术

宽带接入就是将电子设备接入本地ISP。

  1. 非对称数字用户线ADSL

    将现有模拟电话用户线中电话不用的频段给上网使用。

    由于用户使用的时候一般下载远大于上行，因此其分配的频段也是下载多，上行少。（非对称）

    而在通信之前先要调制（ADSL调制解调器）（接入端接单元ATU），我国使用离散多音调DMT，使用频分复用25个子信道为上行，249个为下行。但由于各家情况不同和变化，因此其的调制技术也会对应的自适应，因此不能保证固定的数据率。

    在远端站再解码，与电话信号分离后接入ISP。

    部署：而这整体的改造工程只需要远程进行更改，然后本地将原本的用户线两端加上调制解调器，用户线就变成了数字信号线DSL。非常方便。

    更新：后续又提出了VDSL，更高速率。和ADSL2，数据率更高，能够更加自适应，能够自动纠错等。

    适用范围：普通居民，公司不适合，因为公司需求上行宽带。

  2. 光纤同轴混合网（HFC网）

    将有线电视网的同轴电缆后面的一部分换成光纤，使得其能双向传输并且扩展了传输频带。

    使用机顶盒（电缆调制解调器）作为调制解调器，其调制起来非常复杂，因为是多个用户共用同一条同轴电缆（ADSL每个用户一根），会产生冲突，并且数据率也不稳定，取决于他人。

  3. FTTx（Fiber to The x)

    一般多家用户共用一根光纤（因为需求不高），在多个用户和光路线终端OLT（连接到干线的设备）之间增加一个ODN进行转化。下来的数据广播给每个用户（ONU光网络单元），每个用户根据表示只接受自己的部分，而上传则统一起来向上发送。

    整一个网络称为无源光网络PON。
